类和对象

2017-06-03

学习类和对象的个人笔记

一,两大编程思想

面向过程和面向对象.
面向过程:关注解决问题的步骤
面向对象:关注解决问题参与的事物(对象)

二,类和对象

类,是对一组具有相同特征和行为事物的抽象.
对象: 是具体的.世间万物皆对象.

面向对象更加注重前期的设计, 就是对类的设计,设计类的成员: 特征(属性)和行为(方法)

三,JAVA定义类

​ 访问控制符 类名{
​ //属性(特征)
​ //方法(行为)
​ }

四,使用类创建对象,使用对象访问对象的属性和方法

类名 对象名 = new 类名();

//访问对象的属性
Person person = new Person();
person.name = "张三";
person.age = 18;
person.sex = "男";
//访问对象的方法
person.sayHello();
person.eat();
person.sleep();

属性: 又叫成员属性, 成员变量, 实例变量

五,创建对象的内存分配方式

Java所有new 出来的对象都是在堆区存储的

六,注意事项

  1. 局部变量和成员变量的区别

    • 作用域不一样
    • 内存中的位置不一样
    • 成员变量有默认值, 局部变量没有默认值.
      • 整型属性默认值是0
      • 浮点型属性默认值是0.0
      • 字符型属性默认值是’\u0000’
      • 布尔型属性默认值是false
      • 引用类型属性默认值是null

  2. 属性赋值初始值的方式

    • 属性缺省值

      缺省值就是默认值

    • 定义属性直接显式赋初值

    • 通过对象修改属性内容

七,GC垃圾回收机制

  1. 内存分配
    由JVM自动分配内存空间.
  2. 内存回收
    由JVM的垃圾回收机制自动释放内存空间.
  3. 垃圾回收机制(GC Garbage Collection) 自动将垃圾对象从内存中释放
  4. 垃圾对象指的是不被任何引用指向的对象.
